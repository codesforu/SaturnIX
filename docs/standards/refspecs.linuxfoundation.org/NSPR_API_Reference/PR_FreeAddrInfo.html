PR FreeAddrInfo

Destroys the PRAddrInfo structure returned by PR_GetAddrInfoByName.

Syntax

#include <prnetdb.h>

void PR_EnumerateAddrInfo(PRAddrInfo *addrInfo);

Parameters

The function has the following parameters:

addrInfo
A pointer to a PRAddrInfo structure returned by a successful call to PR_GetAddrInfoByName.

Returns

The function doesn't return anything.
