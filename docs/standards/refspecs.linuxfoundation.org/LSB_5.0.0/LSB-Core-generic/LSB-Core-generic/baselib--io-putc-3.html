_IO_putc

Name

_IO_putc -- alias for putc

Synopsis

int _IO_putc(int __c, _IO_FILE * __fp);

Description

_IO_putc() writes the character __c, cast to an unsigned char, to __fp.

_IO_putc() is not in the source standard; it is only in the binary standard.
