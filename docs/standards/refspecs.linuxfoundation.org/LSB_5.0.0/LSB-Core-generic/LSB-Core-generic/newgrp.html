newgrp

Name

newgrp -- change group ID

Synopsis

newgrp [group]

Description

The newgrp command is as specified in POSIX 1003.1-2008 (ISO/IEC 9945-2009), but with differences as listed below.

Differences

The -l option specified in POSIX 1003.1-2008 (ISO/IEC 9945-2009) need not be supported.
