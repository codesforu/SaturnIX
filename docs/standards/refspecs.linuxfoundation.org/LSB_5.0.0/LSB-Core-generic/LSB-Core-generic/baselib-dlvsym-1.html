dlvsym

Name

dlvsym -- obtain the address of a symbol from a dlopen object

Synopsis

#include <dlfcn.h>

void * dlvsym(void * handle, char * name, char * version);

Description

dlvsym() does the same as dlsym() but takes a version string as an additional argument.
