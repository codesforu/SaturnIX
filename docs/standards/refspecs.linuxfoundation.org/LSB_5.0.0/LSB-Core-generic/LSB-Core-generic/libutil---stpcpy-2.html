__stpcpy

Name

__stpcpy -- alias for stpcpy

Synopsis

#include <string.h>

char * __stpcpy(char * dest, const char * src);

Description

The __stpcpy() function has the same specification as the stpcpy().

__stpcpy() is not in the source standard; it is only in the binary standard.
