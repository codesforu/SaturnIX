_Z7qMallocm

Name

qMalloc -- allocate memory block

Synopsis

#include <QtCore/qglobal.h>

void * qMalloc (size_t size);

Description

The qMalloc() function has the same behavior as the malloc() function described in POSIX 1003.1-2008 (ISO/IEC 9945-2009).
