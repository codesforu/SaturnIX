png_set_strip_16

Name

png_set_strip_16 -- strip 16 bit PNG file to 8 bit depth

Synopsis

#include <png.h>

void png_set_strip_16(png_structp png_ptr);

Description

png_set_strip_16() shall strip the pixels of a PNG stream with 16 bits per channel to 8 bits per channel.
