cupsSetUser

Name

cupsSetUser -- Set the default user name.

Synopsis

#include <cups/cups.h>

void cupsSetUser(const char * user);

Description

Set the default user name.

Pass NULL to restore the default user name.

Return Value

This function does not return a value.
