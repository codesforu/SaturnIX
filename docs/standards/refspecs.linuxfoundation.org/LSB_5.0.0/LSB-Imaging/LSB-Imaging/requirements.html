Chapter 3. Requirements

3.1. Relevant Libraries

The libraries listed in Table 3-1 shall be available on a Linux Standard Base system, with the specified runtime names. This list may be supplemented or amended by an architecture-specific specification.

Table 3-1. Standard Library Names

LibraryRuntime Name
libcupslibcups.so.2
libcupsimagelibcupsimage.so.2
libsanelibsane.so.1

These libraries will be in an implementation-defined directory which the dynamic linker shall search by default.
