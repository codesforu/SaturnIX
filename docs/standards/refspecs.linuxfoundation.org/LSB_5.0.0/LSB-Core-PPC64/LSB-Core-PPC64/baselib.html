Chapter 10. Libraries

An LSB-conforming implementation shall support base libraries which provide interfaces for accessing the operating system, processor and other hardware in the system.

Only interfaces and interface details which are unique to the PPC64 platform are defined here. This section should be used in conjunction with the corresponding section of LSB Core - Generic.

10.1. Program Interpreter/Dynamic Linker

The Program Interpreter shall be /lib64/ld-lsb-ppc64.so.3.
