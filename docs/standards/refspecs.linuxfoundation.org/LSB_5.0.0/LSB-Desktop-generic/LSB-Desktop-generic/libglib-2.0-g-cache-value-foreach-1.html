g_cache_value_foreach

Name

g_cache_value_foreach -- call specified interface for each value in GCache (DEPRECATED)

Synopsis

#include <glib-2.0/glib.h>

void g_cache_value_foreach(GCache * cache, GHFunc func, gpointer user_data);

Description

The interface g_cache_value_foreach() shall behave as described in Glib 2.32 Reference Manual. However, this interface is deprecated and should not be used in newly-written code, because it passes pointers to unspecified internal data structures to func. Use g_cache_key_foreach() instead.
