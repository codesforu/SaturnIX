png_get_bit_depth

Name

png_get_bit_depth -- return image bit_depth

Synopsis

#include <png.h>

png_byte png_get_bit_depth(png_structp png_ptr, png_infop info_ptr);

Description

Returns the image bit_depth.

Return Value

Returns 0 if png_ptr or info_ptr is NULL, bit_depth otherwise.
