Name

qMemCopy -- copy memory area

Synopsis

#include <QtCore/qglobal.h>

void * qMemCopy (void *dest, const void *src, size_t n);

Description

The qMemCopy() function has the same behavior as the memcpy() function described in POSIX 1003.1-2008 (ISO/IEC 9945-2009).
