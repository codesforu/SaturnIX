Chapter 8. Software Installation

8.1. Package Dependencies

The LSB runtime environment shall provide the following dependency.

lsb-desktop-ppc32 

This dependency is used to indicate that the application is dependent on features contained in the LSB Desktop module and contains architecture specific components dependent on this part of LSB Desktop.

This dependency shall have a version of 5.0.
