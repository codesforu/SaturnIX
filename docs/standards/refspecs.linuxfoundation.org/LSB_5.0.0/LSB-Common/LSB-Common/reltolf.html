Chapter 6. Relationship To Other Linux Foundation Specifications

The LSB is foundation of several other specification projects under the umbrella of the Linux Foundation (LF), which build on the interfaces defined here. However, beyond those specifications listed as Normative References in the various LSB specification volumes, this specification has no dependencies on other LF projects.
